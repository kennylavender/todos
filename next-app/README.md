# todos

## Description

A practice, demo app, using [TodoMVC](http://todomvc.com/) as a loose starting point.

## Development

```
$ npm run dev
```

## Unit Tests

Single run

```
$ npm run test-unit
```

Run on changes

```
$ npm run test-unit-watch
```

## Missing some tests

Doing some exploratory coding, so not doing TDD on everything at the moment.
